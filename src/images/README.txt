CELEBRITY IMAGES SETUP GUIDE
=============================

Place the following image files in this folder:

1. michelle.jpg - Michelle Chong's photo
2. nelson.jpg - Nelson Mandela's photo  
3. taylor.jpg - Taylor Swift's photo
4. singapore_uncle.jpg - Singapore Uncle's photo

IMAGE REQUIREMENTS:
- Format: .jpg, .jpeg, .png, or .webp
- Recommended size: 400x400 pixels (square format)
- The images should be clear headshots/portraits
- File names must match exactly (lowercase)

EXAMPLE FILE STRUCTURE:
src/images/
├── michelle.jpg
├── nelson.jpg
├── singapore_uncle.jpg
├── taylor.jpg
└── README.txt

Once you add the images, they will automatically appear when:
1. User selects an opponent from the dropdown
2. The debate starts and the AI begins speaking
3. The avatar will show a speaking indicator when active

The system will fall back to the default doctor avatar if images are missing.
